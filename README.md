# SMM学习-员工管理系统

#### 介绍
基于SMM框架下的员工管理系统

#### 软件架构
Spring + SpringMVC + Mybatis

Mysql 5.7

Tomcat 9


#### 安装教程

1.  新建数据库news, 导入sql文件夹下的sql文件, 并在database.properties文件下修改本地的数据库信息;
2.  确保Tomcat环境;
